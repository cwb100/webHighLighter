# Web Highlighter

一个 Chrome / Edge 浏览器扩展，用来给网页内容划重点。

## 功能

- 在网页中选中文本后，选择不同颜色高亮
- 高亮结果自动保存
- 刷新页面后恢复高亮
- 点击已高亮文本后可弹出操作菜单，支持删除或调整颜色
- 支持预设颜色和自定义颜色
- 支持删除单条标记或清空整页标记
- 提供独立控制台，查看所有已划线网页和对应标记

## 项目结构

- `manifest.json`：扩展配置
- `content.js`：网页内高亮与恢复逻辑
- `background.js`：点击扩展图标时打开控制台
- `dashboard.html` / `dashboard.js` / `dashboard.css`：独立管理页

## 安装方式

1. 打开 Chrome 或 Edge。
2. 进入 `chrome://extensions` 或 `edge://extensions`。
3. 开启“开发者模式”。
4. 选择“加载已解压的扩展程序”。
5. 选中这个项目目录。

## 使用方法

1. 在任意网页中选中一段文字。
2. 出现浮动工具条后，选择一种颜色。
3. 文字会被高亮并自动保存。
4. 再次点击已高亮的文字，可弹出菜单删除或调整颜色。
5. 点击浏览器扩展图标，打开控制台查看所有记录。

## 说明

- 当前版本面向普通网页文章、资讯页和长文页面。
- 只要页面内容结构变化不太大，高亮会在刷新后恢复。
- 控制台里可以删除单条标记，也可以清空整页标记。
